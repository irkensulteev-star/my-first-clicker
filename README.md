# My First Clicker Game 🕹️

A simple, interactive, and stylish web-based clicker game built from scratch to learn the fundamentals of front-end web development.

## ✨ Features
* **Interactive Clicking:** Click the "Hit Me" button to earn points instantly.
* **Upgrade System:** Save up 10 points to buy an upgrade that automatically generates points over time.
* **Modern Dark UI:** Clean, minimalist dark-themed design built with custom CSS.
* **Responsive Layout:** Centered interface that scales nicely on different screen sizes.

## 🛠️ Technologies Used
* **HTML5** – Structured the game elements and layouts.
* **CSS3** – Styled the buttons, fonts, and dark theme colors.
* **JavaScript (ES6)** – Engineered the game logic, event listeners, conditional statements (`if/else`), and interval timers (`setInterval`).

## 🧠 What I Learned
Through building this project entirely on my own, I mastered:
1. Connecting HTML elements to JavaScript using `document.getElementById`.
2. Managing game states with variable scopes (`let` and `const`).
3. Handling user interactions via asynchronous event listeners.
4. Structuring clean, reusable layout styles in CSS.
5. Deploying web apps to production using **Git** and **GitHub Desktop**.
